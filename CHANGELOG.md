# 更新日志

## [0.3.0] - 2026-03-13

### Added

- 新增求片系统，支持电影、整剧、单季求片
- 新增后台求片管理和求片设置页面
- 新增 MoviePilot 推送和可选的自动推送 MP
- 新增用户求片次数限制，支持按天 / 周 / 月配置
- 新增 Emby Webhook 入库通知，可自动完成求片工单

### Changed

- 用户中心改为多页面结构，保留原有导航样式
- 卡密续费与兑换记录合并为同一页面
- 自动闭环升级为“定时扫描 + Emby Webhook”双兜底

### Fixed

- 修复电影 `library.new` webhook 在 Emby 索引延迟时无法自动完成工单的问题

## [0.2.4] - 2026-03-12

### Added

- 用户管理增加调整到期功能

### Fixed

- Emby 设置保存时会校验地址和 Key 是否可用
- 修复基础用户权限模板弹窗内容左右边框被挡住的问题
- 修复用户列表表格在手机端挤压显示的问题

## [0.2.3] - 2026-03-11

### Added

- 版本更新检查
- 线路管理页，支持增删改查和排序
- 线路支持显示开关和白名单可见
- 用户页增加服务器线路卡片，支持复制线路
- 用户页增加上次活动、最近 30 天观看时长、设备和 IP 信息

### Fixed

- 修复 Bugs

## [0.2.0] - 2026-03-11

### Added

- 新增 `播放统计` 数据来自 Emby Playback Reporting 插件
- 新增 Emby `/Sessions` 实时会话标签页，支持在线用户、正在播放、暂停中和当前会话列表
- 新增全站播放时长排行、用户 IP / 设备数量排行、全站设备使用排行
- 新增客户端占比、设备分布占比、热门影片 / 热门剧集排行
- 新增多 IP / 多设备 / 高频切换基础风控排行
- 新增用户画像页，支持单用户播放趋势、播放汇总、设备 / IP 画像
- 新增用户管理到播放画像的跳转链路
- 新增用户权限配置
- 新增用户权限模版以及批量同步
- 新增从本地恢复用户到 Emby
- 新增 Emby 简易探针

### Fixed

- 修复 Bug
