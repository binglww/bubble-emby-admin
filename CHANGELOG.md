# 更新日志

## [0.2.3] - 2026-03-11

### Added

- 版本更新检查
- 线路管理页，支持增删改查和排序
- 线路支持显示开关和白名单可见
- 用户页增加服务器线路卡片，支持复制线路
- 用户页增加上次活动、最近 30 天观看时长、设备和 IP 信息

### Fixed

- 修复 Bugs

## [0.2.0] - 2026-03-11

### Added

- 新增 `播放统计` 数据来自 Emby Playback Reporting 插件
- 新增 Emby `/Sessions` 实时会话标签页，支持在线用户、正在播放、暂停中和当前会话列表
- 新增全站播放时长排行、用户 IP / 设备数量排行、全站设备使用排行
- 新增客户端占比、设备分布占比、热门影片 / 热门剧集排行
- 新增多 IP / 多设备 / 高频切换基础风控排行
- 新增用户画像页，支持单用户播放趋势、播放汇总、设备 / IP 画像
- 新增用户管理到播放画像的跳转链路
- 新增用户权限配置
- 新增用户权限模版以及批量同步
- 新增从本地恢复用户到 Emby
- 新增 Emby 简易探针

### Fixed

- 修复 Bug
